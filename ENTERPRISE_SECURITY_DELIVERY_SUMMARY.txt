ENTERPRISE SECURITY POLICY FEATURES
================================================================================

The EnterpriseSecurityPolicy model supports 30+ configurable settings:

AUTHENTICATION (MFA)
- require_mfa: Enable/disable multi-factor authentication
- mfa_grace_period_days: Grace period before MFA enforcement
- require_hardware_keys: Require hardware security keys

PASSWORD POLICY
- password_min_length: Minimum password length (default: 16)
- password_require_uppercase: Uppercase letter requirement
- password_require_lowercase: Lowercase letter requirement
- password_require_numbers: Number requirement
- password_require_special_chars: Special character requirement
- password_expiry_days: Password expiry (default: 90)
- password_history_count: Previous passwords to remember

SESSION MANAGEMENT
- session_timeout_minutes: Session inactivity timeout
- session_concurrent_limit: Max concurrent sessions per user
- require_vpn_for_admin: Require VPN for admin access

DATA PROTECTION
- encryption_at_rest: Enable encryption at rest
- encryption_algorithm: Algorithm used (AES-256, etc.)
- tls_minimum_version: Minimum TLS version (1.3, etc.)

AUDIT & LOGGING
- enable_audit_logging: Enable audit logging
- audit_log_retention_days: How long to retain logs
- log_all_access: Log all access events
- log_all_changes: Log all changes

ACCESS CONTROL
- enforce_least_privilege: Least privilege enforcement
- require_approval_for_privileged_access: PAM workflow
- privileged_access_approval_timeout_hours: Approval timeout

NETWORK SECURITY
- enforce_network_segmentation: Network segmentation required
- enforce_zero_trust: Zero trust architecture
- require_vpn_for_remote_access: VPN requirement

VULNERABILITY MANAGEMENT
- enable_vuln_scanning: Vulnerability scanning enabled
- vuln_scan_frequency_days: Scan frequency
- auto_patch_critical: Auto-patch critical vulnerabilities
- patch_window_hours: Maintenance window for patching

BACKUP & RECOVERY
- enable_backups: Backup enabled
- backup_frequency_hours: Backup frequency
- backup_retention_days: Backup retention
- rto_hours: Recovery Time Objective
- rpo_minutes: Recovery Point Objective

COMPLIANCE
- annual_audit_required: Annual audit requirement
- penetration_testing_frequency_months: Pentest frequency
